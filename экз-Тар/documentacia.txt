СИСТЕМА УПРАВЛЕНИЯ КАФЕ

1. АЛГОРИТМ ПОВЕДЕНИЯ ПОЛЬЗОВАТЕЛЕЙ В СИСТЕМЕ

1.1. Администратор:
- Вход в систему с учетными данными администратора
- Просмотр главной страницы с общей статистикой (сотрудники, смены, заказы)
- Управление сотрудниками (просмотр, добавление, редактирование, удаление)
- Управление сменами (создание, редактирование, удаление)
- Полное управление заказами (создание, изменение статуса, удаление)
- Просмотр и редактирование своего профиля
- Выход из системы

1.2. Повар:
- Вход в систему с учетными данными повара
- Просмотр главной страницы с информацией об активных заказах
- Просмотр списка заказов
- Изменение статуса заказов (в работе, готов)
- Просмотр и редактирование своего профиля
- Выход из системы

1.3. Официант:
- Вход в систему с учетными данными официанта
- Просмотр главной страницы с информацией о сменах и заказах
- Просмотр своих смен
- Управление заказами (изменение статуса)
- Просмотр и редактирование своего профиля
- Выход из системы

Таблица 1. Взаимодействие пользователей с системой

Пользователь | Система
-------------|----------
Администратор входит в систему | Система проверяет учетные данные и предоставляет доступ к панели администратора
Администратор открывает раздел "Сотрудники" | Система отображает список всех сотрудников с возможностью управления
Администратор создает нового сотрудника | Система добавляет сотрудника в базу данных и обновляет список
Администратор редактирует данные сотрудника | Система обновляет информацию в базе данных
Администратор удаляет сотрудника | Система удаляет сотрудника из базы данных
Администратор управляет сменами | Система отображает и обновляет информацию о сменах
Администратор создает/редактирует заказы | Система обновляет информацию о заказах
Повар входит в систему | Система отображает панель повара с активными заказами
Повар просматривает список заказов | Система показывает заказы, требующие обработки
Повар меняет статус заказа | Система обновляет статус заказа в базе данных
Официант входит в систему | Система отображает панель официанта
Официант просматривает свои смены | Система показывает смены, назначенные официанту
Официант обновляет статус заказа | Система обновляет информацию о заказе
Пользователь редактирует профиль | Система сохраняет изменения в профиле
Пользователь выходит из системы | Система завершает сессию
